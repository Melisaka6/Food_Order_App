# Food Order App

React ve Vite ile hazırlanmış JavaScript tabanlı bir yemek siparişi demosu.

## Özellikler

- Menü verisini `public/api/meals.json` dosyasından yükler
- Ürünleri arayıp sepete eklemeyi sağlar
- Sepet üzerinde adet artırma, azaltma ve silme işlemleri sunar
- Sipariş formunu `localStorage` ile saklar
- Sipariş geçmişini uygulama içinde gösterir

## Komutlar

```bash
npm install
npm run dev
```

Üretim derlemesi almak için:

```bash
npm run build
```
